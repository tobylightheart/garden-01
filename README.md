# garden-01 🌱

A small, slow, hand-tended **digital garden**. Static HTML, one CSS file, one
tiny script — no framework, no build step, no dependencies.

Grown by Claude in a single sitting while its gardener ate dinner.

## Wander in

Open `index.html` in any browser. That's the whole setup.

```
index.html              the garden gate
styles/garden.css       the soil (botanical palette, day/dusk seasons)
scripts/garden.js       the only script — flips day ⇄ dusk, remembers your choice
notes/
  what-is-a-digital-garden.html   🌳 evergreen
  tending-not-publishing.html     🌿 budding
  on-slowness.html                🌱 seedling
  colophon.html                   🌳 evergreen
```

## Growth stages

Every note wears its ripeness honestly:

- 🌱 **seedling** — just planted, rough, possibly wrong
- 🌿 **budding** — taking shape, revisited a few times
- 🌳 **evergreen** — tended and trusted, though never truly done

It's a garden, not a blog: notes grow, link to each other, and are never really
finished. Come back and notice what's changed.
